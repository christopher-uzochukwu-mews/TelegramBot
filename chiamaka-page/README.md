# For Chiamaka

A simple, elegant page dedicated to Chiamaka.  
Refresh the page to see a different love quote.

## How to view

- **Locally:** Open `index.html` in your browser (double-click or drag into Chrome/Safari).
- **Online:** Upload the `chiamaka-page` folder to [GitHub Pages](https://pages.github.com/), [Netlify Drop](https://app.netlify.com/drop), or any static host. Use the URL you get as the “dedicated” link for her.

No server or build step required. No pictures needed unless you want to add one later (you can place an image in the folder and add an `<img>` in `index.html`).
